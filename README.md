# Miniguia de Estudo com NotebookLM – Python aplicado ao mercado financeiro

## Sobre o projeto
Este projeto foi desenvolvido com o objetivo de criar um caderno temático no NotebookLM a partir de fontes abertas sobre **Python aplicado ao mercado financeiro**. A proposta consistiu em reunir conteúdos em vídeo sobre programação, análise de dados e aplicações práticas em finanças, definir objetivos de estudo, testar prompts estratégicos e transformar esse material em um miniguia de revisão.

## Objetivo
Utilizar IA como ferramenta de aprendizagem ativa, combinando:
- curadoria de fontes;
- pensamento crítico;
- organização do conhecimento;
- criação de materiais reutilizáveis para estudo.

## Tema escolhido
**Python aplicado ao mercado financeiro**

## Objetivos de estudo
O foco deste projeto foi compreender como Python pode ser utilizado na prática dentro do mercado financeiro, especialmente em atividades como:
- análise de dados financeiros;
- automação de processos e relatórios;
- construção de indicadores e visualizações;
- organização e tratamento de bases de dados;
- apoio à tomada de decisão em finanças;
- desenvolvimento de lógica aplicada ao contexto profissional.

## Metodologia
A atividade foi desenvolvida em etapas:
1. seleção de fontes abertas;
2. organização do material no NotebookLM;
3. definição dos objetivos de estudo;
4. criação de perguntas estratégicas;
5. testes de variações de prompts;
6. registro das respostas geradas com base nas fontes;
7. consolidação do conteúdo em um miniguia de estudo.

## Uso do NotebookLM
O NotebookLM foi utilizado como apoio para:
- resumir os conteúdos das fontes;
- explicar conceitos de forma mais didática;
- responder perguntas estratégicas sobre Python e finanças;
- estruturar revisões e glossários;
- transformar conteúdos dispersos em um material de estudo mais claro e reutilizável.

## Entregáveis
Este repositório contém:
- descrição do projeto;
- referências utilizadas;
- prompts reutilizáveis;
- direcionamento de estudo com foco em Python aplicado ao mercado financeiro.

## Resultados
O projeto permitiu organizar uma trilha inicial de aprendizado sobre Python aplicado ao mercado financeiro, utilizando IA como suporte para estudo ativo. A proposta ajudou a transformar várias fontes abertas em um material mais estruturado, com foco em aplicação prática, clareza conceitual e utilidade para futuras revisões.

## Observação
As fontes utilizadas no projeto são abertas e foram selecionadas para apoiar a construção do caderno temático no NotebookLM.
